# Shoe App
This is a Shoe Store App made using react. The user can search for shoes and filter based on shoe filters like
category, color, price, etc.


## Note:
It is only a front-end application with no backend. Sample Shoes data is
stored in a data.jsx file.
This project implements the following concepts in react:
- React Basics (Conditional Rendering, Map filter etc.)
- React hooks such as useState.
- Context API (For providing the states to all other components avoiding prop drilling)
